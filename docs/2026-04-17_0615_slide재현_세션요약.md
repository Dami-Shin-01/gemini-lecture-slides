## Decisions

- **단일 HTML 파일 결정** — 외부 JS 프레임워크(React/Vue) 없이 바닐라 JS + CSS만으로 구성. 로컬/정적 호스팅/오프라인 시연 모두 지원. (결과물 기반 확정)
- **4챕터 구조** (AI소개 → 프롬프트 → 문서작성 → 데이터분석) — 이론→실습→챌린지 흐름을 슬라이드 타입으로 분리(content/compare/diagram/practice).
- **챕터별 컬러 토큰** (ch1 블루 #5B8DEF, ch2 레드 #D4564E, ch3 그린 #4CAF50, ch4 오렌지 #FF9800) — 진행도 바도 현재 슬라이드의 챕터색으로 동기 전환.
- **슬라이드 종횡비 16:9 고정** — `width: min(100vw, calc(100vh * 16 / 9))`로 창 크기에 관계없이 비율 유지.
- **반응형 타이포그래피** — 모든 텍스트 크기를 `clamp()`로 정의(예: `--text-h1: clamp(36px, 4vw, 54px)`). 투사 환경 변화에도 적응.
- **발표자 노트 인라인** — 각 `<section>`에 `data-notes` 속성으로 삽입, N키로 패널 토글. 별도 파일 관리 불필요.
- **슬라이드 타입 8종** — cover / chapter / content / compare / diagram / practice / divider / ending. 타입별 CSS 클래스(`slide--*`)로 완전 분리.
- **키보드 UX** — `← → ↑ ↓ Space`, `Home/End`, `N`(notes), `G`(goto), `Esc`(close). 마우스는 좌/우 절반 클릭.
- **모바일 가로 강제** — `@media (orientation: portrait) and (max-width: 900px)`에서 `body::before`로 가로 회전 안내 오버레이.
- **애니메이션 배치 규칙** — bullet-list는 80ms 간격 순차(`nth-child` delay), flow-step은 100ms 간격 pop-in(`--ease-spring`).

## 디자인 시스템 요약

- **폰트**: Pretendard Variable (CDN: `cdn.jsdelivr.net/gh/orioncactus/pretendard/dist/web/variable/pretendardvariable.css`) + mono fallback (Fira Code / D2Coding / Consolas)
- **타입 스케일**: `--text-display(48-72)` → `--text-h1(36-54)` → `--text-h2(24-38)` → `--text-h3(18-26)` → `--text-body(15-22)` → `--text-caption(12-16)` → `--text-label(10-14)`
- **공간 스케일**: `--space-xs(8) → sm(16) → md(24) → lg(40) → xl(64) → 2xl(96)`
- **Radius**: `sm 6 / md 12 / lg 20 / pill 999`
- **Easing**: `--ease-out(0.16,1,0.3,1)` / `--ease-spring(0.34,1.56,0.64,1)`
- **Duration**: slide 420ms / fade 250ms
